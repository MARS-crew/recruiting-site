# 마스외전 5기 모집 사이트

```
해당 프로젝트의 패키지 관리자는 yarn을 사용합니다. 아래의 명령어로 yarn을 전역으로 설치해주세요.

npm install -g yarn
```

### RUN

```
1. git clone
2. cd recruiting-site
3. yarn
4. yarn dev
```
